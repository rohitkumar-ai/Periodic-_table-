# 🧪 Interactive Periodic Table

An interactive, responsive periodic table of elements built with HTML, CSS, and JavaScript.

## ✨ Features

- Click on any element to display detailed information
- Color-coded element categories (noble gases, alkali metals, etc.)
- Responsive design - works on desktop, tablet, and mobile
- Smooth hover animations
- Displays atomic number, symbol, name, and atomic mass



## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (Vanilla)
